## Исключаем из поиска (префильтр)

Если в названии лота есть эти слова — пропускай, не отправляй в бота:

**Запрещённые платформы:**
- 1С, 1С:Предприятие, 1С: Бухгалтерия
- Bitrix, Битрикс, Битрикс24, 1C-Битрикс
- WordPress
- SharePoint
- Joomla, Drupal
- Moodle *(если требуется именно на Moodle — исключаем)*

**Оборудование / on-site / физика:**
- поставка оборудования
- поставка компьютеров
- поставка серверов
- монтаж
- пусконаладка
- пуско-наладочные работы
- техническое обслуживание *(чистый maintenance)*
- сервисное обслуживание
- текущий ремонт
- капитальный ремонт
- строительство

**Регулируемое / секретное:**
- СКЗИ
- ФСТЭК
- гостайна
- секретно
- ДСП
- работы для закрытых объектов
- работы на режимном объекте

**Нестандартные deliverables:**
- обучение персонала (если это единственный предмет закупки)
- повышение квалификации
- сертификация сотрудников
- аудит *(чистый аудит без разработки)*
- консалтинг *(чистый консалтинг)*

## Как интерн работает

1. Заходит на goszakup.gov.kz → раздел "Поиск объявлений".
2. Ищет по ключевым словам из секции **"Ищем"** (по одному слову за раз).
3. Фильтрует по статусу "Опубликован" / "Приём заявок".
4. Открывает карточку лота → смотрит название и краткое описание.
5. Если в названии есть слова из **"Исключаем"** — пропускает.
6. Если нет — скачивает техзадание (обычно "Техническая спецификация" или "Техническое задание").
7. Отправляет документ в бота.
8. После запроса бота — копирует ссылку на лот и отправляет в чат.
9. Получает подтверждение от бота, переходит к следующему.

## Приоритет поиска

Начинать с самых перспективных по частоте попаданий в GO:

1. `лендинг`, `посадочная страница`, `сайт-визитка`
2. `разработка сайта`, `разработка веб-сайта`
3. `разработка CRM`, `разработка личного кабинета`
4. `разработка чат-бота`, `telegram-бот`
5. `парсер`, `агрегатор`, `сбор данных`
6. `API`, `интеграция систем`
7. `дашборд`, `BI`
8. `веб-портал`, `веб-приложение`
9. `автоматизация`, `workflow`

## Заметки на будущее

- Позже можно автоматизировать через API goszakup.gov.kz (есть REST/SOAP, документация на сайте портала).
- skstore.kz — для квази-госсектора (Samruk-Kazyna), там обычно более приличные бюджеты.
- Коммерческие b2b тендеры — отдельный трек, там меньше жёстких требований по юрлицу/сертификации.
